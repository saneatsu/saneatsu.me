# Saneatsu Portfolio

Welcome to the Storybook for Saneatsu's portfolio website.

This project follows Feature-Sliced Design architecture principles.

## Layers

- **app**: Application initialization, routing, and global providers
- **views**: Full page components  
- **widgets**: Large, composite UI blocks
- **features**: User interaction flows and business logic
- **entities**: Business domain objects and data
- **shared**: Reusable utilities and components
